# Lesson 4.1 — MITRE Container Framework Demo

> From the [Certified Kubernetes Security Specialist (CKS) Video Course](https://www.pearsonitcertification.com/store/certified-kubernetes-security-specialist-cks-video-9780138296476)

## Objective

Understand how to map the MITRE ATT&CK framework to Kubernetes clusters and identify security threats in container environments.

## Prerequisites

- Kubernetes cluster access
- Familiarity with MITRE ATT&CK framework
- Basic knowledge of container security concepts

## Steps

### Step 1: Review MITRE ATT&CK Framework Mapping

Reference the MITRE ATT&CK framework mapping to Kubernetes cluster initial access techniques:

https://kloudle.com/academy/part-1-mapping-the-mitre-att-ck-framework-to-your-kubernetes-cluster-initial-access/

### Step 2: Explore Kubernetes Security Vulnerabilities

Review the Kubernetes Goat project to understand common security vulnerabilities and attack scenarios in Kubernetes:

https://github.com/madhuakula/kubernetes-goat

## Verification

Verify understanding by:
- Identifying initial access vectors in your Kubernetes cluster
- Mapping detected attack patterns to MITRE ATT&CK techniques
- Documenting security gaps in your environment
